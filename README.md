# Miro Zukina | Full Stack Stack Software Developer Portfolio

A responsive, single-page portfolio website showcasing modern UI design elements, neon typography glows, interactive theme configuration capabilities, and integrated form processing logic. 

**Live Demo:** [https://mirozukina.github.io/Profolio/](https://mirozukina.github.io/Profolio/)

## 🚀 Tech Stack & Core Features

- **Frontend Architecture:** Semantic HTML5, Modular CSS3, and Vanilla JavaScript.
- **Visual Design:** Fluid responsive layout mechanics, integrated dark theme canvas, dynamic drop-shadow neon pulse animations, and interactive variable theme switching.
- **UX & Utilities:** Dynamic scroll sync tracking active nav links, non-blocking asynchronous mobile navigation bar, and structural regex email schema field validation.
